迅雷VIP账号自动获取
==============


>>使用方式
>>>双击 迅雷VIP账号自动获取.cmd
>>>每行显示一组用户名和密码
>>>在头部边框，右击编辑->标记，选中所需要的用户名，按Enter即可复制，密码同理
>>>（如果之后时间有限会对这部分简化设计）

>>dll
>>>提供一个免T破解DLL，源自网络，对使用不当造成的任何损失，本人概不负责。
>>>将DLL，覆盖迅雷安装目录program下面的同名文件即可
